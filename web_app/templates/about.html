About INIDS

Project Overview

The Intelligent Network Intrusion Detection System (INIDS) is an advanced machine learning-based solution for detecting cyber attacks in network traffic. This system demonstrates the practical application of multiple ML algorithms in cybersecurity.

Key Features

  • 5+ Machine Learning Models (Random Forest, Gradient Boosting, Decision Tree, AdaBoost, MLP)
  • Binary and Multiclass Classification
  • Real-time Prediction with Confidence Scores
  • Comprehensive Analytics Dashboard
  • Batch Processing Capabilities
  • ROC Curve Analysis
  • Feature Importance Visualization

Dataset

NSL-KDD Dataset: An improved version of the KDD'99 dataset, widely used for network intrusion detection research. Contains 125,973 training samples and 22,544 test samples with 41 features.

Attack Categories

  • DoS: Denial of Service attacks
  • Probe: Surveillance and probing
  • R2L: Remote to Local attacks
  • U2R: User to Root attacks

Technologies Used

Python, Scikit-learn, Flask, Pandas, Matplotlib, Bootstrap, NumPy
